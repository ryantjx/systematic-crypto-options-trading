# LLM Instructions

This folder contains instructions, context, and guidelines for the LLM assistant to reference when working on this project.

## Purpose

Store any files here that you want the LLM to have access to, including:
- Project-specific guidelines and conventions
- Architecture documentation
- Coding standards
- Domain knowledge and context
- Task lists and priorities
- Any other reference material

## Usage

The LLM has full read/write access to this folder as part of the workspace. You can:
1. Add markdown files with specific instructions
2. Include code examples or templates
3. Document project decisions and rationale
4. Provide domain-specific context

The LLM can reference these files to better understand your project requirements and preferences.
